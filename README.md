# 时间戳转换工具 / Timestamp Converter

在线时间戳与北京时间互转工具，支持秒/毫秒，一键复制，深色模式，响应式布局。

An online timestamp ↔ Beijing time converter. Supports seconds/milliseconds, one-click copy, dark mode, responsive layout.

**在线体验 / Demo**：<https://dsun6652-oss.github.io/easy-timestamp-tool>

## 功能 / Features

- 时间戳 → 北京时间 / Timestamp → Beijing Time
- 日期时间 → 时间戳 / Date & Time → Timestamp
- 支持秒 / 毫秒 / Seconds & Milliseconds
- 一键复制 / One-click copy
- 清空 / Clear
- 响应式（手机也能用）/ Responsive (mobile-friendly)
- 中英文切换 / Chinese & English
